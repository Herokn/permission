## 3. 技术 Spec

### 3.1 RoleService 接口

```java
public interface RoleService {
    RoleDO getById(Long id);
    RoleDO getByCode(String code);
    void save(RoleDO roleDO);
    void updateById(RoleDO roleDO);
    void removeById(Long id);
    IPage<RoleDO> page(RoleQueryDTO queryDTO);
}
```

### 3.2 RoleServiceImpl 实现要点

```
getByCode(code): LambdaQueryWrapper.eq(code).one()

page(queryDTO):
  LambdaQueryWrapper
    .like(code != null, RoleDO::getCode, code)
    .like(name != null, RoleDO::getName, name)
    .eq(roleScope != null, RoleDO::getRoleScope, roleScope)
    .eq(roleDomain != null, RoleDO::getRoleDomain, roleDomain)
    .eq(status != null, RoleDO::getStatus, status)
    .orderByDesc(RoleDO::getGmtCreate)
```

### 3.3 RolePermissionService 接口

```java
public interface RolePermissionService {
    /** 按角色 ID 查询权限编码列表 */
    List<RolePermissionDO> listByRoleId(Long roleId);

    /** 按角色 ID 删除所有关联（逻辑删除） */
    void deleteByRoleId(Long roleId);

    /** 批量保存 */
    void saveBatch(List<RolePermissionDO> list);

    /** 按权限编码统计引用数（权限点删除前检查用） */
    long countByPermissionCode(String permissionCode);

    /** 判断角色是否包含某权限（鉴权用） */
    boolean exists(Long roleId, String permissionCode);
}
```

### 3.4 RolePermissionServiceImpl 实现要点

```
listByRoleId(roleId):
  LambdaQueryWrapper.eq(RolePermissionDO::getRoleId, roleId).list()

deleteByRoleId(roleId):
  LambdaQueryWrapper.eq(RolePermissionDO::getRoleId, roleId)
  rolePermissionMapper.delete(wrapper)  // MyBatis-Plus 逻辑删除

saveBatch(list):
  使用 MyBatis-Plus ServiceImpl.saveBatch() 或循环 insert

countByPermissionCode(permissionCode):
  LambdaQueryWrapper.eq(RolePermissionDO::getPermissionCode, permissionCode).count()

exists(roleId, permissionCode):
  LambdaQueryWrapper
    .eq(RolePermissionDO::getRoleId, roleId)
    .eq(RolePermissionDO::getPermissionCode, permissionCode)
    .count() > 0
```

### 3.5 RoleManager 接口

```java
public interface RoleManager {
    RoleVO createRole(CreateRoleDTO dto);
    RoleVO updateRole(Long id, UpdateRoleDTO dto);
    void deleteRole(Long id);
    PageResult<RoleVO> listRoles(RoleQueryDTO queryDTO);
    void assignPermissions(Long roleId, AssignRolePermissionDTO dto);
    List<String> getRolePermissions(Long roleId);
}
```

### 3.6 RoleManagerImpl 实现要点

#### createRole

```
1. code 唯一性: roleService.getByCode(dto.code) → 非空则抛 ROLE_CODE_EXISTS
2. 枚举校验: RoleScopeEnum.fromCode(dto.roleScope), RoleDomainEnum.fromCode(dto.roleDomain)
3. Converter 转 DO, 设置 status = ENABLED
4. roleService.save(do)
5. 返回 RoleVO
```

#### updateRole

```
1. existing = roleService.getById(id) → null 则抛 ROLE_NOT_FOUND
2. 若 dto.roleScope 非空 → 枚举校验
3. 若 dto.roleDomain 非空 → 枚举校验
4. 若 dto.status 非空 → 枚举校验
5. 更新非 null 字段
6. roleService.updateById(existing)
7. 返回 RoleVO
```

#### deleteRole

```
@Transactional(rollbackFor = Exception.class)
1. existing = roleService.getById(id) → null 则抛 ROLE_NOT_FOUND
2. 用户引用检查: userRoleService.countByRoleId(id) > 0 → 抛 ROLE_USED_BY_USER
3. 清理角色-权限关联: rolePermissionService.deleteByRoleId(id)
4. 逻辑删除角色: roleService.removeById(id)
```

**注意**：deleteRole 需要注入 UserRoleService（跨模块依赖，Plan-04 创建）。

#### assignPermissions

```
@Transactional(rollbackFor = Exception.class)
1. 校验角色: role = roleService.getById(roleId)
   → null 抛 ROLE_NOT_FOUND
   → DISABLED 抛 ROLE_DISABLED
2. 校验每个权限点:
   for code in dto.permissionCodes:
     perm = permissionService.getByCode(code)
     → null 抛 PERMISSION_NOT_FOUND
     → DISABLED 抛 PERMISSION_DISABLED
3. 全量覆盖: rolePermissionService.deleteByRoleId(roleId)
4. 批量插入: 构建 List<RolePermissionDO>, rolePermissionService.saveBatch(list)
```

#### getRolePermissions

```
1. list = rolePermissionService.listByRoleId(roleId)
2. return list.stream().map(RolePermissionDO::getPermissionCode).collect(toList())
```

### 3.7 DTO 定义

### 3.10 RoleController

```java
@RestController
@RequestMapping("/roles")
public class RoleController {

    @PostMapping

    public ApiResponse<RoleVO> update(@PathVariable Long id, @RequestBody @Valid UpdateRoleDTO dto)

    public ApiResponse<PageResult<RoleVO>> list(RoleQueryDTO queryDTO)

    public ApiResponse<List<String>> getRolePermissions(@PathVariable Long roleId)
}
```
