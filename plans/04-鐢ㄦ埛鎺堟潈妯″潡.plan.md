实现用户-角色分配/移除、用户直接权限授予(ALLOW)/排除(DENY)/移除、用户授权详情查询。依赖 RoleService 和 PermissionService 进行校验。

## 3. 技术 Spec

### 3.1 UserRoleService 接口

```java
public interface UserRoleService {
    void save(UserRoleDO userRoleDO);
    UserRoleDO getById(Long id);
    void removeById(Long id);

    /**
     * 唯一性校验：同一用户+角色+项目 是否已存在
     * 注意 projectId 为 null 时需要用 IS NULL 查询
     */
    boolean exists(String userId, Long roleId, String projectId);

    /**
     * 查询用户在指定项目 + 全局的角色列表
     * 条件：user_id = ? AND (project_id = ? OR project_id IS NULL)
     */
    List<UserRoleDO> listByUserIdAndProjectId(String userId, String projectId);

    /** 按角色 ID 统计引用数（角色删除前检查用） */
    long countByRoleId(Long roleId);
}
```

### 3.2 UserRoleServiceImpl 实现要点

```
exists(userId, roleId, projectId):
  LambdaQueryWrapper
    .eq(UserRoleDO::getUserId, userId)
    .eq(UserRoleDO::getRoleId, roleId)
  if projectId == null:
    .isNull(UserRoleDO::getProjectId)
  else:
    .eq(UserRoleDO::getProjectId, projectId)
  return count() > 0

listByUserIdAndProjectId(userId, projectId):
  LambdaQueryWrapper
    .eq(UserRoleDO::getUserId, userId)
    .and(w -> {
      if projectId != null:
        w.eq(UserRoleDO::getProjectId, projectId).or().isNull(UserRoleDO::getProjectId)
      else:
        w.isNull(UserRoleDO::getProjectId)
    })
  注意：当 projectId 为 null 时，只查全局（projectId IS NULL）
  当 projectId 非空时，查项目级 + 全局

countByRoleId(roleId):
  LambdaQueryWrapper.eq(UserRoleDO::getRoleId, roleId).count()
```

### 3.3 UserPermissionService 接口

```java
public interface UserPermissionService {
    void save(UserPermissionDO userPermissionDO);
    UserPermissionDO getById(Long id);
    void removeById(Long id);

    /** 唯一性校验 */
    boolean exists(String userId, String permissionCode, String effect, String projectId);

    /** 查询用户在指定项目 + 全局的直接权限列表 */
    List<UserPermissionDO> listByUserIdAndProjectId(String userId, String projectId);

    /** 按权限编码统计引用数（权限点删除前检查用） */
    long countByPermissionCode(String permissionCode);

    // ========== 鉴权专用方法 ==========

    /** 查找 DENY 记录（项目级 + 全局），返回第一条匹配 */
    UserPermissionDO findDeny(String userId, String permissionCode, String projectId);

    /** 查找 ALLOW 记录（项目级 + 全局），返回第一条匹配 */
    UserPermissionDO findAllow(String userId, String permissionCode, String projectId);
}
```

### 3.4 UserPermissionServiceImpl 实现要点

```
exists(userId, permissionCode, effect, projectId):
  LambdaQueryWrapper
    .eq(userId).eq(permissionCode).eq(effect)
  if projectId == null: .isNull(projectId)
  else: .eq(projectId)
  return count() > 0

listByUserIdAndProjectId(userId, projectId):
  同 UserRoleService 的 projectId 匹配逻辑

countByPermissionCode(permissionCode):
  LambdaQueryWrapper.eq(permissionCode).count()

findDeny(userId, permissionCode, projectId):
  LambdaQueryWrapper
    .eq(userId).eq(permissionCode)
    .eq(effect, "DENY")
    .and(w -> w.eq(projectId, projectId).or().isNull(projectId))
    .last("LIMIT 1")
  return selectOne(wrapper)

findAllow: 同上，effect = "ALLOW"
```

### 3.5 UserAuthManager 接口

```java
public interface UserAuthManager {
    UserRoleVO assignUserRole(String userId, AssignUserRoleDTO dto);
    void removeUserRole(String userId, Long userRoleId);
    UserPermissionVO assignUserPermission(String userId, AssignUserPermissionDTO dto);
    void removeUserPermission(String userId, Long userPermissionId);
    UserAuthorizationVO getUserAuthorizations(String userId, String projectId);
}
```

### 3.6 UserAuthManagerImpl 实现要点

#### assignUserRole

```
1. role = roleService.getById(dto.roleId)
   → null 抛 ROLE_NOT_FOUND
   → DISABLED 抛 ROLE_DISABLED
2. GLOBAL 角色校验:
   if role.roleScope == GLOBAL && dto.projectId != null → 抛 GLOBAL_ROLE_NO_PROJECT
3. 唯一性校验:
   userRoleService.exists(userId, dto.roleId, dto.projectId) → true 抛 USER_ROLE_EXISTS
4. 构建 UserRoleDO(userId, roleId, projectId), save
5. 构建 UserRoleVO（含 roleCode, roleName 关联信息）返回
```

#### removeUserRole

```
1. userRole = userRoleService.getById(userRoleId)
   → null 抛 USER_ROLE_NOT_FOUND
2. 校验 userId 匹配: userRole.userId != userId → 抛 USER_ROLE_NOT_FOUND（安全性）
3. userRoleService.removeById(userRoleId)
```

#### assignUserPermission

```
1. perm = permissionService.getByCode(dto.permissionCode)
   → null 抛 PERMISSION_NOT_FOUND
   → DISABLED 抛 PERMISSION_DISABLED
2. 枚举校验: PermissionEffectEnum.fromCode(dto.effect)
3. 唯一性校验:
   userPermissionService.exists(userId, dto.permissionCode, dto.effect, dto.projectId)
   → true 抛 USER_PERMISSION_EXISTS
4. 构建 UserPermissionDO(userId, permissionCode, effect, projectId), save
5. 构建 UserPermissionVO（含 permissionName 关联信息）返回
```

#### removeUserPermission

```
1. record = userPermissionService.getById(userPermissionId)
   → null 抛 USER_PERMISSION_NOT_FOUND
2. 校验 userId 匹配
3. userPermissionService.removeById(userPermissionId)
```

#### getUserAuthorizations

```
1. 查询用户角色列表:
   userRoles = userRoleService.listByUserIdAndProjectId(userId, projectId)
2. 构建角色详情列表:
   for each userRole:
     role = roleService.getById(userRole.roleId)
     if role == null || role.status != ENABLED → skip
     permCodes = rolePermissionService.listByRoleId(role.id)
       .stream().map(getPermissionCode).collect(toList())
     构建 UserRoleDetailVO(userRoleId, roleId, roleCode, roleName, projectId, permissions)
3. 查询用户直接权限:
   userPerms = userPermissionService.listByUserIdAndProjectId(userId, projectId)
4. 按 effect 分组:
   allowList = filter(effect == ALLOW) → 转 UserPermissionVO（关联查权限名称）
   denyList = filter(effect == DENY) → 转 UserPermissionVO
5. 组装 UserAuthorizationVO(userId, projectId, roles, directPermissions{allow, deny})
```

### 3.9 UserAuthWebConverter

```java
@Mapper
public interface UserAuthWebConverter {
    UserAuthWebConverter INSTANCE = Mappers.getMapper(UserAuthWebConverter.class);
    UserRoleVO toUserRoleVO(UserRoleDO userRoleDO);
    UserPermissionVO toUserPermissionVO(UserPermissionDO userPermissionDO);

```

### 3.10 UserAuthController

```java
@RestController
@RequestMapping("/users")
public class UserAuthController {

    @PostMapping("/{userId}/roles")
    public ApiResponse<UserRoleVO> assignRole(@PathVariable String userId,
                                               @RequestBody @Valid AssignUserRoleDTO dto)

    @DeleteMapping("/{userId}/roles/{userRoleId}")
    public ApiResponse<Void> removeRole(@PathVariable String userId,
                                         @PathVariable Long userRoleId)

    @PostMapping("/{userId}/permissions")
    public ApiResponse<UserPermissionVO> assignPermission(@PathVariable String userId,
                                                           @RequestBody @Valid AssignUserPermissionDTO dto)

    @DeleteMapping("/{userId}/permissions/{userPermissionId}")
    public ApiResponse<Void> removePermission(@PathVariable String userId,
                                               @PathVariable Long userPermissionId)

    @GetMapping("/{userId}/authorizations")
    public ApiResponse<UserAuthorizationVO> getAuthorizations(@PathVariable String userId,
                                                               @RequestParam(required = false) String projectId)
}
```
